¡Bienvenido a MorfiPolo!

MorfiPolo es la aplicación oficial para gestionar y votar por los menús del comedor. Con esta app podrás:

• Ver el menú del día de forma rápida y sencilla
• Votar por tu opción favorita del menú
• Consultar los menús de la semana completa
• Recibir notificaciones recordatorias para no perderte la hora de votación
• Acceder a tu historial de votos

Características principales:
✓ Interfaz intuitiva y fácil de usar
✓ Widget para ver el menú directamente desde tu pantalla de inicio
✓ Notificaciones personalizables
✓ Sincronización en tiempo real con el servidor
✓ Modo claro optimizado para una mejor experiencia visual

Esta es la primera versión de MorfiPolo. Estamos trabajando constantemente para mejorar la aplicación y agregar nuevas funcionalidades.

¡Gracias por usar MorfiPolo!
